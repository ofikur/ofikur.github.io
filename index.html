Keahlian Teknologi

Languages & Core

  • HTML5

    Pondasi utama untuk membangun struktur dan konten halaman web.

  • CSS3

    Menghias dan menata tampilan halaman web agar menarik dan responsif.

  • JavaScript (ES6+)

    Menambahkan interaktivitas dan logika dinamis pada sisi klien.

  • TypeScript

    Superset dari JavaScript yang menambahkan static typing untuk skalabilitas.

Frameworks & Libraries

  • React

    Library utama untuk membangun antarmuka pengguna yang reaktif dan berbasis komponen.

  • Next.js

    Framework React yang mendukung rendering sisi server (SSR) dan optimasi performa.

  • Vue.js

    Framework progresif untuk membangun antarmuka pengguna yang interaktif.

  • GSAP

    Tools animasi profesional untuk membuat interaksi yang kaya dan performan.

Styling

  • Tailwind CSS

    Framework CSS berbasis utility-first untuk prototyping dan desain cepat.

  • Bootstrap

    Framework CSS populer untuk pengembangan web yang responsif dan mobile-first.

  • Sass/SCSS

    Preprocessor CSS yang menambahkan fitur seperti variabel, nesting, dan mixins.

  • Framer Motion

    Library animasi untuk React yang memudahkan pembuatan transisi UI yang kompleks.

  • CSS Modules

    Menjaga style tetap lokal per komponen untuk menghindari konflik class global.

Tools & Platforms

  • Vite

  • Figma
